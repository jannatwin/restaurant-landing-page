Saffron Table Fine Dining Restaurant
A premium, modern single-page website for Saffron Table, an elegant fine dining restaurant established in 2018. The project showcases modern web design principles with a responsive layout, beautiful typography, elegant dark aesthetic themes, and micro-interactions.
🍽️ Features
Responsive Navigation: Includes a mobile-friendly slide-out drawer menu toggled by a hamburger button.
Hero Banner: High-resolution photography backdrop with overlay text and CTA buttons.
Quick Info Strip: Quick-access details for operating hours, physical location, and direct reservations.
Our Story / About Section: Highlight of the restaurant's legacy, culinary stats, and chef values.
Signature Menu Grid: A clean, grid-based menu layout with price details, chef highlights, and high-quality food photography.
Guest Testimonials: Elegant reviews slider/grid showing community feedback with star ratings.
Table Reservation Form: An interactive form layout for table booking, fully responsive.
Clean Structure & Semantic HTML: Accessible markup structure utilizing standard HTML5 semantic tags.
📁 File Structure
The project consists of the following key files:
p2.html: The markup structure of the single-page website including sections, forms, and inline navigation drawer script.
p2.css: Styled layout, responsive design media queries, color schemes, and component styling.
🚀 How to Run Locally
Since the application is built using standard Vanilla HTML and CSS, you can run it directly:
Double-click p2.html to open it in any web browser.
Alternatively, use a local development server like VS Code's "Live Server" extension to serve the files locally.
